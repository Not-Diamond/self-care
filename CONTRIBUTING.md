# Contributing to Self-Care

Thank you for your interest in Self-Care!

## Project Status

Self-Care is publicly available to use and explore, but it is not in active development. This repository is maintained on a best-effort basis, and we may not respond to issues or pull requests on a consistent timeline.

## Bug Reports

[Open an issue](https://github.com/Not-Diamond/self-care/issues) with the **Bug report** template. A good bug report includes:

- Steps to reproduce the problem
- Expected vs actual behavior
- Trace file format used (OTEL JSON, Claude Code trace, etc.)
- Plugin version (`/self-care:help` to check)
- Claude Code version and OS

## Pull Requests

Small, well-scoped fixes may be considered, but unsolicited pull requests are not guaranteed review or merge. We may close issues or PRs that are out of scope, stale, or not aligned with current maintenance priorities.

## Security Vulnerabilities

Please see our [Security Policy](./SECURITY.md) for how to report vulnerabilities privately.

## Code of Conduct

Please read our [Code of Conduct](./CODE_OF_CONDUCT.md). We expect all participants to follow it.
